Installation
===================

``fiscai`` is currently hosted on PyPI. It required python >= 3.7.

You can simply install fiscai from PyPI with the following command:

.. code:: shell

    pip install fiscai

You can also install with the newest version through GitHub:

.. code:: shell

    pip install -U git+https://github.com/HansBug/FiscAI.git@main

of ``fiscai`` should be shown.
